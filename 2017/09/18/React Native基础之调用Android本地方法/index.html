有几点需要说明：

  • getName返回的NativeModule名字，在后面JaveScript代码中是用来表示此类的，在js中我们可以通过NativeModule.ToastExample来访问此类

  • 为了能够在js中调用Android的本地方法，需要被调用的方法必须加上@ReactMethod注释，作为js和java中的桥梁方法，该方法返回类型需要是void，访问权限为public

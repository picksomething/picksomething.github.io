结尾

这就是箭头函数带来的两个比较大的改变，但是使用箭头函数的时候也有几点需要注意的：

  • this对象的指向是可变的，但是在箭头函数中，它是固定的，函数体内的this对象，绑定定义时所在的对象，而不是使用时所在的对象
  • 不可以当作构造函数，不可以使用new命令
